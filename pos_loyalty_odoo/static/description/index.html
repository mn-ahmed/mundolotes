POS- Point Of Sales Loyalty Rewards and Redeem Management Odoo Apps

POS Loyalty and Rewards Redeem odoo apps is used to give POS loyalty redemption points for every purchase to your customers from point of sales screen.Customer can also redeem this loyalty points for other purchase from the point of sale screen. Every single purchase from the point of sale records the Loyalty Rewards based on configuration setup on point of sale backend and those rewards will be redeemed on POS order from point of sale touch screen order easily. Reward redeems visible on applied point of sales order so it will be helpful for see history of the reward point redemption.

Features

Define Loyalty Points

Define Loyalty based on Pos product category and purchase amount.

Earns loyalty points

Customer can earns loyalty points with his/her every purchase in POS.

Loyalty Point Configuration

Apply loyalty points based on the loyalty configuration rule.

Use loyalty points

Customers can use loyalty points as a discount for other purchase orders in POS.
Demo Request Video Tutorial

There are two options for loyalty (i) Purchase Amount and (ii) POS Product Categories.

Loyalty Based on Purchase Amount

POS Loyalty Configuration

Under Point of Sale > Loyalty Management > Loyalty Configuration menu the user can create a loyalty configuration, User needs to select a product for loyalty, and enter the amount.

Users can also configure redeem rule for loyalty configuration.

Loyalty on POS

On point of sale screen user can see added loyalty point on the purchase amount(select customer first).

Loyalty Point on POS Receipt

Users can see earned loyalty points in point of sale order receipts.

Loyalty History

Under the Point of Sale > Loyalty Management > Loyalty History menu, the user can see generated loyalty history for credit points.

Customer's Loyalty Points

Users can see customer's loyalty points from point of sale screen.

Users can also see added customers loyalty points in the customer view in the smart button or under the "Sales & Purchase" tab.

Loyalty Based on POS Product Categories

POS Loyalty Configuration

Under Point of Sale > Loyalty Management > Loyalty Configuration menu user can create a loyalty configuration, User needs to select product categories for loyalty.

Users can also configure redeem rule for loyalty configuration.

POS Product Categories Configuration

Users need to enter "Amount for Loyalty Points" for each pos product category.

Loyalty on POS

On point of sale screen, users can see added loyalty points as per product category configurations.

Loyalty Point on POS Receipt

Users can see earned loyalty points in point of sale order receipts.

Loyalty History

Under the Point of Sale > Loyalty Management > Loyalty History menu, the user can see generated loyalty history for credit points.

Customer's Loyalty Points

Users can see customer's loyalty points from point of sale screen.

Users can also see added customer's loyalty points in the customer view in the smart button or under the "Sales & Purchase" tab.

Loyalty Redemption

Users can also redeem earned loyalty points from point of sale by clicking the "Redeem" button on the screen.

Enter the number of points to Redeem, Here user can see the total loyalty points and total loyalty amount number of points, and total amount according to the redemption rule.

Users can see on clicking the "Redeem" button loyalty product with a minus amount added.

Loyalty History

Users can see generated loyalty history for debit points.
