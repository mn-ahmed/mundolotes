Odoo Discount Coupons and Gift voucher for Point of Sale and Website Odoo Apps

This odoo apps helps to manage Coupons and Gift voucher for POS and Website both. Some of the business manage pos gift coupon and website gift voucher both. So in this case same coupon and discount voucher need to create once and use for both point of sale and website. This apps also helps to create discount Coupons and Gift voucher from the Point of Sale as well as from the Odoo backend and this Coupons and Gift voucher can be redeem from the website too, So same Coupons discount and Gift voucher can be use on Odoo Shop and Odoo Point of sale.

Create Gift Coupon From Back-end

Under Point of Sale > Gift Coupons > All Gift Coupons menu user can create and see all created gift coupons.

USer have option to create gift coupons with "Amount Type" as "Discount or Percentage", "Allow for Specific Customer", "Allow for Specific Products", and "Allow for Specific Categories" for "Ecommerce Categories Or POS Categories".

Here user can see gift coupon with "Allow for Specific Products" option.

Gift Product

User can select "Product" on gift coupon, For that "Coupon Product" need to enable and also "Available in POS" and published on website.

Print Gift Coupon

User can print gift coupon report in PDF format by clicking "PRINT" button.

Voucher Code Option For Website

User need to enable "Voucher Code" feature on website to apply coupon code.

Use Coupon From Website

User can apply voucher code by entering and clicking "Apply" button.

User can see voucher code applied and voucher amount deducted.
